# strawberryphp

[![Latest Version on Packagist][ico-version]][link-packagist]
[![Software License][ico-license]](LICENSE.md)

A simplified PHP API router

## Install

Via Composer

``` bash
$ composer require strawberrykit/strawberryphp
```
